# black-bar

Draw a black rectangle on an image.

## Usage

```
Usage: black-bar [options] <file>

obscure part of an image with a black bar

Options:
  -V, --version            output the version number
  -x <OFFSET>              x offset (default: 5)
  -y <OFFSET>              y offset (default: 10)
  -w, --width <WIDTH>      width of black bar (default: 67)
  -h, --height <HEIGHT>    height of black bar (default: 21)
  -o, --output <FILENAME>  output result to filename (default: "test.png")
  -h, --help               display help for command
```
